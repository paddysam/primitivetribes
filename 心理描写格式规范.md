# 心理描写格式规范

## 核心规则
主角的内心活动一律加引号，当内心台词处理，和对话格式一致。

## 格式示范

### ✅ 正确写法（加引号，独立成段）

"完了。"

"他妈的，捡回一条命。"

林华心里迅速盘算："六个，都带着石矛，领头的那个身上有伤。打不过，但不一定非要打。"

"别瞎想，肯定有合理的解释。"

### ❌ 错误写法（不加引号，混在叙述里）

完了。

他妈的，捡回一条命。

他心里迅速盘算：六个，都带着石矛，领头的那个身上有伤。打不过，但不一定非要打。

别瞎想，肯定有合理的解释。

## 三种心理描写的格式

### 闪念型（一句话，独立成段）
```
　　林华的瞳孔猛地收缩。

　　"完了。"
```

### 盘算型（两三句，可以带叙述引导）
```
　　林华扫了一眼对面那群人，心里迅速盘算："六个，都带着石矛，领头的那个身上有伤。打不过，但不一定非要打。"
```

### 情绪型（一小段，不超过五句）
```
　　"妈，你儿子可能到不了了。"

　　他使劲咬了一下舌头，把这个念头压下去。
```

## 注意事项
- 内心台词加引号后要独立成段，不要和动作描写挤在同一行
- 连续多句内心活动，每句各占一段
- 叙述性的思维（"他注意到""他发现"）不加引号，只有内心"说出来的话"才加
- 不要在引号内写太长，超过两行就拆开或改用叙述带出

## 区分标准

加引号的（内心在"说话"）：
- "打不过。"
- "这人八成是首领。"
- "我们不会是要被吃了吧？"

不加引号的（叙述性思维）：
- 林华注意到了这个细节。
- 他的脑子飞速转了一圈。
- 一种说不清的寒意从脚底往上蹿。
